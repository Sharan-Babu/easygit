The following is the format for adding new intents.
Syntax:
##
Intent name and response: <intent_name>=<response>.

Replace the values within '<>' accordingly.

##
Intent name and response: git_clone=git clone <repo_url>.
How to clone my repo?
How to get the repo on my local machine?
What is the syntax/command for cloning a repository in git?

##
Intent name and response: git_branch=git branch <branch_name>.
How to create a new branch?
New branch syntax?
